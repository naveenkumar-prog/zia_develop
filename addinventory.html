Procedure:

  • Download the template by clicking on the "Download template here" button below.
  • Go to the "Upload Inventory" sheet in the downloaded file and enter inventory data into it.
  • For adding details of product, enter data into the "Upload Product" sheet with the corresponding serial number. Every row of data in the details sheet must have the serial number of the corresponding product
  • After listing all products and their data into the downloaded template, upload the file below (under the ".xlsx file" label).
  • In the Excel Sheet the should be filled according to the following Rules:
  • There are 3 Types in the Fields(Columns):-
  • TYPE 1 : Compulsory Fields, Data must be filled in these fields
  • TYPE 2 : Optional Fields, Data may be filled or not in these fields but not compulsory
  • TYPE 3 : These Fileds are also Optional. But if any one of the fields under any set is filled then remaining all fields under Corresponding set also must be filled
  • TYPE 4 : Old value will be overwritten if you change this, But this field is Compulsory. So, add '0' to keep old value


    • The following are the fields with their type. Please fill the fields according to their type as mentioned above:-
  • TYPE 1 : Fields are Product_Id,Seller_Name,Seller_Mobile,Seller_Address,Quantity, Quantity_Unit, Seller_Price_Per_Unit_Without_tax_and_offer
  • TYPE 2 : Field is IGST
  • TYPE 3(Set-1) : Fields are CGST, SGST
  • TYPE 3(Set-2) : Fields are MRP, Zila_Offer, CP_Commission
  • TYPE 4 : Field is Threshold
  • Click on "Upload" button and wait until all the products are uploaded.
  • If there is an error in any one the inventory items while bulk upload, the error will be shown off. But Other Items(Correct items) will get added
  • So, while make correction, try not to add those items which are get already added
  • And Please Make sure that all Comulsory fields have been added in each row before upload
  • Keep the file name as ZilaUploadInventory.
  • Keep the sheet in file as it is don't rename/delete them .
  • If Sheet name and the fields get changed, then items won't get add .

Please Follow the Procedure Correctly, Otherwise the Items won't get add !!
